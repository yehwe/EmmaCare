# Health Monitor
A portfolio-grade ESP32 project integrating MAX30100 pulse oximeter, DS18B20 
temperature sensor, WiFi HTTP upload, and SIM900D SMS alerts.
## Features- Heart rate, SpO2, and temperature monitoring- Automatic SMS alerts for critical conditions
## Setup
1. Connect sensors and SIM900D per `Simulation.jpg`
2. Modify `Config.h` with your WiFi and server info
3. Upload firmware using Arduino IDE or PlatformIO
